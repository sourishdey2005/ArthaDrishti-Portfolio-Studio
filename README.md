# Black-Litterman Portfolio Optimizer

An interactive web application for portfolio optimization using the Black-Litterman model with investor views.

## Features

- 📊 **Real-time Data**: Fetch stock data from Yahoo Finance
- 🎯 **Investor Views**: Add absolute and relative views with confidence levels
- 📈 **Black-Litterman Model**: Combines market equilibrium with investor views
- ⚡ **Portfolio Optimization**: Mean-variance optimization with constraints
- 📉 **Efficient Frontier**: Visualize the efficient frontier
- 📊 **Interactive Visualizations**: Dynamic charts using Plotly
- 💾 **Export Results**: Download results as CSV
